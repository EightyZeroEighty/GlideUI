Design System Base Style Guide v1.0

These elements are intended to be themed for specific projects.

Layout & Grid System

A 12-column responsive grid system for structuring page layouts. Uses a flexbox-based approach.

Grid Overview

The grid is composed of containers, rows, and columns. Content should be placed within columns, and columns must be immediate children of rows.

  • .glide-container: Wraps rows. Manages max-width and centering. Has horizontal padding.
  • .glide-row: Wrapper for columns. Uses flexbox to arrange columns.
  • .glide-col-*: Defines column widths. From .glide-col-1 to .glide-col-12.
  • Responsive Columns: .glide-col-sm-*, .glide-col-md-*, .glide-col-lg-*, .glide-col-xl-*, .glide-col-xxl-* for different breakpoints.

Breakpoints

Prefix Min-width Container Max-width
(none) - Mobile First 0px 100%
sm 576px (--breakpoint-sm) 540px (--container-max-width-sm)
md 768px (--breakpoint-md) 720px (--container-max-width-md)
lg 992px (--breakpoint-lg) 960px (--container-max-width-lg)
xl 1200px (--breakpoint-xl) 1140px (--container-max-width-xl)
xxl 1400px (--breakpoint-xxl) 1320px (--container-max-width-xxl)

Default Gutter Width: var(--grid-gutter-width) (24px)
